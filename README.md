# Company-Bankruptcy-Prediction
Company Bankruptcy Prediction using Random Forest

## Overview
This project predicts corporate bankruptcy using machine learning techniques.

## Dataset
Financial ratios of companies with a binary target:
- `1` → Bankrupt
- `0` → Non-bankrupt

## Methods Used
- Exploratory Data Analysis (EDA)
- Class imbalance handling using **Oversampling**
- Random Forest Classifier
- Hyperparameter tuning with GridSearchCV
- Confusion Matrix, Classification Report

## Technologies
- Python
- Pandas, NumPy
- Scikit-learn
- Imbalanced-learn
- Matplotlib, Seaborn
